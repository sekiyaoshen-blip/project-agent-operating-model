## Handoff Rewrite And Size Budget

Keep this file compact and rewrite it as current truth instead of appending forever.

Recommended limits:

- Current State: 3-5 bullets
- Main-Thread Relevant Changes: 3-7 bullets
- Contract Changes: only changed or newly discovered contracts
- Risks / Blockers: only active risks
- Verification: latest meaningful checks only
- Next Local Actions: 3-5 bullets

Remove or archive:

- Completed details already accepted by the main thread
- Superseded risks
- Raw logs
- Long implementation narratives
- Repeated file lists
- Old debug notes
- Local implementation detail not needed by the main thread
